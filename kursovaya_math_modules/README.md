# Coursework Project

Topic 10: modules for mathematical functions in Python.

## Project idea

The project is a desktop application `Math Modules Explorer` built with object-oriented design and the standard Python library. It demonstrates practical work with several mathematical modules and approaches:

- `math` for transcendental and trigonometric functions
- `statistics` for descriptive statistics
- `decimal` for accurate financial calculations
- `ast` for safe parsing of user expressions
- `tkinter` for the graphical interface

## Features

- evaluate mathematical expressions
- solve quadratic equations with real or complex roots
- calculate descriptive statistics for a numeric dataset
- compute compound interest with `Decimal`
- plot the graph of a function `y = f(x)` on a canvas

## Run

```bash
python main.py
```

## Tests

python -m unittest discover -s tests -v
